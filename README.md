
# Proyek Akhir Sistem Mikrokontroler

## Judul
Sistem Lampu Tidur Otomatis Berbasis ESP32 Menggunakan Sensor LDR dan Blynk

## Nama
Neng Nova Siti Fathonah

## Deskripsi
Proyek ini menggunakan ESP32, sensor LDR, dan LED untuk membuat lampu tidur otomatis. Sistem juga terhubung ke platform Blynk sebagai implementasi Internet of Things (IoT).

## Komponen
- ESP32
- Sensor LDR
- LED
- Resistor
- Blynk
